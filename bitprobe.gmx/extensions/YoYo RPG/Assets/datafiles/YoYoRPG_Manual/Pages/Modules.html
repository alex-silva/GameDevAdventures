Script Modules

The approach taken to create this engine has been to use a series of modular scripts to create general functions which can be applied to the various different objects in the game. In this way you can have a sign-post that talks to you, or a chicken that can sell you goods or specific backgrounds that animate. Note that not all of the scripts that are grouped together are "modules". Most of them are essential game engine scripts that you should not normally need to touch - only those outlined below are those that you should build from to change the game to suit your needs to start with.

The links below will give you an overview of the main modules that are available to you and show you examples of instances already in the game which use them - however should you need more practical examples of how to use them, please see the sections on Adding Instances and Creating An Area.
